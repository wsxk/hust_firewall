# hust_firewall
华中科技大学 2019级 网络安全课程设计 状态检测防火墙

# 环境
ubuntu 16.04 和 ubuntu 20.04 下测试，能正确完成编译和运行。

# 安装方法
```shell
  sudo su
  make
  insmod wxk_firewall.ko
  cd user_app
  make
  ./uapp
```
客户端用法请参考代码

# 缺陷
1. nat没写（懒）
2. 需要先设计拒绝规则再尝试连入（在设计逻辑时，默认情况下已存在连接，直接无视规则表；在添加规则时，并没有进行已有连接的删除操作；理由很简单，不想写了，摆烂....，其实想改也很简单）
